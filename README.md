
# Zomato Data Analysis

## 📊 Overview
This project analyzes Zomato restaurant data to identify trends, customer behavior, and business insights.

## 🛠 Tools Used
- Excel


## 🔍 Key Insights
- Identified top-performing cities and restaurants
- Analyzed customer preferences and ratings
- Found trends in pricing and cuisine

## 📁 Files Included
- Dataset (Excel)
- Dashboard (Excel)

## 🚀 Outcome
This project helps understand market trends and supports data-driven decision making.
